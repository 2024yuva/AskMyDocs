└──────────────────────────────────────────────────────────────────────────────┘

STEP 1: Get FREE Groq API Key (2 minutes)
  1. Visit: https://console.groq.com/keys
  2. Sign up (FREE, no credit card)
  3. Create API key
  4. Copy your key

STEP 2: Add Key to .env File
  1. Open .env file
  2. Replace: GROQ_API_KEY=your_groq_api_key_here
  3. With your actual key: GROQ_API_KEY=gsk_your_actual_key

STEP 3: Run the Project
  1. Double-click: ingest_docs_groq.bat
  2. Double-click: start_api_groq.bat
  3. Double-click: start_ui.bat
