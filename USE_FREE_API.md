# 🆓 How to Use FREE APIs with AskMyDocs

## 🚀 Quick Start with Groq (Recommended - FREE & FAST!)

### Step 1: Get Free Groq API Key (2 minutes)

1. Visit: **https://console.groq.com**
2. Sign up (free, no credit card required)
3. Go to: **https://console.groq.com/keys**
4. Click "Create API Key"
5. Copy your key

### Step 2: Setup Project

**Double-click:** `setup_groq.bat`

This will:
- Install required packages (langchain-groq, sentence-transformers)
- Configure the project for free APIs

### Step 3: Add Your API Key

Edit `.env` file and add your Groq API key:
```
GROQ_API_KEY=gsk_your_actual_groq_key_here
```

### Step 4: Run the Project

1. **Double-click:** `ingest_docs.bat`
2. **Double-click:** `start_api.bat`
3. **Double-click:** `start_ui.bat`

That's it! You're using 100% FREE APIs! 🎉

## 🔄 Alternative: 100% Local with Ollama (No API Key!)

If you want ZERO API keys and complete privacy:

### Step 1: Install Ollama

1. Download: **https://ollama.com/download**
2. Install Ollama for Windows
3. Open terminal and run:
   ```bash
   ollama pull llama3.2
   ollama pull nomic-embed-text
   ```

### Step 2: Configure Project

I can create an Ollama configuration for you. Just say "use Ollama"!

### Pros of Ollama:
- ✅ 100% free forever
- ✅ No API keys needed
- ✅ Complete privacy (runs locally)
- ✅ No rate limits
- ✅ Works offline

### Cons of Ollama:
- ❌ Requires ~8GB RAM
- ❌ Slower than cloud APIs
- ❌ Need to download models (~4GB)
