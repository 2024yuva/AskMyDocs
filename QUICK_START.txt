🎯 WHAT IS THIS?
   Chat with your documents using AI! Upload PDFs, Markdown, or text
   files and ask questions. Get accurate answers with source citations.

💡 TECHNOLOGY USED:
   ✓ Groq API (FREE Llama 3.3 70B) - LLM generation
   ✓ Sentence-Transformers (FREE) - Local embeddings
   ✓ Cohere (FREE tier) - Reranking
   ✓ LangChain - RAG pipeline
   ✓ FastAPI - Backend API
   ✓ Streamlit - Web UI
   ✓ ChromaDB - Vector database
   ✓ BM25 - Keyword search

🚀 HOW TO RUN:

   Step 1: Install dependencies (first time only)
   ─────────────────────────────────────────────
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt

   Step 2: Configure API keys
   ─────────────────────────────────────────────
   Edit .env file and add your Groq API key:
   GROQ_API_KEY=your_key_here
   
   Get FREE key at: https://console.groq.com/keys

   Step 3: Run the project
   ─────────────────────────────────────────────
   Double-click: RUN_PROJECT.bat
   
   Or in terminal: .\RUN_PROJECT.bat

   Step 4: Use the application
   ─────────────────────────────────────────────
   Open browser: http://localhost:8501
   
   1. Upload documents (sidebar)
   2. Click "🔄 Ingest Documents"
   3. Ask questions in chat!

📁 PROJECT STRUCTURE:
   RUN_PROJECT.bat     ← Run this to start!
   README.md           ← Full documentation
   .env                ← Your API keys
   requirements.txt    ← Dependencies
   app/                ← Application code
   docs/               ← Put your documents here
   tests/              ← Test suite
   eval/               ← Evaluation pipeline

🔧 TROUBLESHOOTING:
   • API won't start? Check if port 8000 is in use
   • Ingestion fails? Verify Groq API key in .env
   • No answers? Make sure you ingested documents first
