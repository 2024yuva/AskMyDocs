⚠️  IMPORTANT: Your OpenAI API key has exceeded its quota!

┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 0: Fix Your API Key (REQUIRED)                                         │
└──────────────────────────────────────────────────────────────────────────────┘

Option A: Add Credits to Existing Key
  1. Visit: https://platform.openai.com/account/billing
  2. Add payment method
  3. Add credits ($5 minimum recommended)
  
Option B: Get New API Key
  1. Visit: https://platform.openai.com/api-keys
  2. Create new key
  3. Edit .env file and replace OPENAI_API_KEY value

┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 1: Ingest Documents                                                    │
└──────────────────────────────────────────────────────────────────────────────┘

  📁 Double-click: ingest_docs.bat
  
  This will process the 3 markdown files in docs/ folder
  ⏱️  Takes ~30 seconds

┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 2: Start API Server                                                    │
└──────────────────────────────────────────────────────────────────────────────┘

  🖥️  Double-click: start_api.bat
  
  API will run at: http://localhost:8000
  Keep this window open!

┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 3: Start Streamlit UI                                                  │
└──────────────────────────────────────────────────────────────────────────────┘

  🎨 Double-click: start_ui.bat
  
  Browser will open automatically at: http://localhost:8501
  Keep this window open too!

┌──────────────────────────────────────────────────────────────────────────────┐
│ STEP 4: Use the Application                                                 │
└──────────────────────────────────────────────────────────────────────────────┘

  In the Streamlit UI:
  
  ✅ Upload your own documents (PDF, MD, TXT)
  ✅ Click "Ingest Documents" to process them
  ✅ Ask questions in the chat interface
  ✅ View citations and source documents
  
  Example questions:
  • "What is RAG?"
  • "How does hybrid retrieval work?"
  • "Explain cross-encoder reranking"

   → Add credits to your OpenAI account

   → Make sure start_api.bat is running

   → Run ingest_docs.bat first

   → Close other instances or change port
